# Retail Analytics Dataset

## Overview

The **Retail Analytics Dataset** is a relational retail dataset designed for Analytics Engineering, Data Engineering, and Data Warehousing projects. It simulates real-world retail business operations by combining customer, product, sales, inventory, supplier, and operational data into an interconnected data model.

This dataset is intended for building end-to-end modern data platforms using technologies such as AWS, Snowflake, dbt, Airflow, Spark, SQL, and BI tools. It provides realistic business entities, complex relationships, and dimensional modeling scenarios suitable for enterprise-scale analytics.

---

## Dataset Highlights

- Relational retail business dataset
- Multiple interconnected tables
- Customer, Product, Sales, Inventory, Supplier, and Store domains
- Supports Star Schema and Snowflake Schema modeling
- Suitable for Medallion Architecture (Bronze, Silver, Gold)
- Ideal for ETL / ELT pipelines
- Designed for Data Warehouse projects
- Supports Business Intelligence and Reporting
- Ready for Analytics Engineering use cases
