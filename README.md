# 📊 Data Engineering Datasets

A curated collection of free, open-source, and real-world datasets for practicing **Data Engineering**, **Analytics Engineering**, **SQL**, **Python**, **dbt**, **Apache Spark**, **Snowflake**, **BigQuery**, **Amazon Redshift**, **Azure Synapse Analytics**, **DuckDB**, **Polars**, **ETL/ELT pipelines**, **Data Warehousing**, **Data Lakes**, **Data Modeling**, and modern cloud data platform projects.

---

## 🎯 Purpose

This repository is a centralized collection of high-quality datasets for learning, experimenting, and building real-world Data Engineering and Analytics Engineering projects. Instead of searching for datasets every time you start a new project, you can find them all in one place.

Whether you're learning SQL, building ETL pipelines, modeling data warehouses, practicing dbt, working with Apache Spark, or exploring modern cloud data platforms, this repository provides datasets across multiple industries and use cases to support hands-on learning.
